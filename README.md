# Notas
Ejericio de la tabla de multiplicar
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Describe la base de la multimplicación     [número] [requerido]
  -l, --listar   Describe si se imprime la lista en consola
                                         [booleano] [requerido] [defecto: false]
  -h, --hasta    Describe hasta que número multilplicar la base
                                              [número] [requerido] [defecto: 10]
```
